FAQ

Device questions teams ask first

Short, practical answers.

Do MITE-ECO devices work without MITE?
They can send telemetry via standard protocols, but MITE is where the value is: logic, workflows, reporting, audit trails, and outcomes.
Can we use our existing sensors or gateways?
Yes. MITE-ECO is optional. If you already have hardware, we adapt the integration layer and normalize data into the same operational model.
How do you handle constrained networks?
We design for real conditions: buffering, predictable payload formats, and stable operation even when connectivity is limited.
Can we customize what is monitored per product/site?
Yes — that’s the point. You can create as many rules and monitoring regimes as you need without rebuilding the system.
